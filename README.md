Modeliranje i razvoj informacionog sistema teretane

Samostalni projekat u kom je korišćeno znanje stečeno na fakultetskim kursevima Baze podataka, Modeliranje informacionih sistema i Razvoj informacionih sistema.
Cilj ovog projekta je automatizacija poslovanja teretane i implementacija tog sistema.
Ovaj projekat je realizovan korišćenjem Spring Boot-a i Spring Security-a. 

Projekat sadrži osmišljavanje celokupnog sistema tako da pomogne raznim korisnicima, uvođenje funkcionalnosti i povezivanje korisnika.

U fajlu opis.odt se nalazi tekstualni opis sistema: koje uloge postoje u sistemu i koje funkcionalnosti su im omogućene.
Takođe dijagram slučaja korišćenja koji je napravljen u alatu Bouml opisuje funkcionalnosti aplikacije.
Pored fajla teretanaSql.sql u kom se nalazi kod za importovanje baze podataka u MySQL, tu je i relacioni model šeme baze podataka.
U .xml fajlovima u projektu su dodati svi potrebni dependency-i.

Da bi se pokrenula aplikacija potrebno je da skinuti foldere TeretanaJPA.zip i TeretanaWEB.zip i pokrenuti aplikaciju online na serveru(korišćen je server Apache Tomcat v9.0).
